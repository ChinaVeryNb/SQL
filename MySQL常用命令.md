## MySQL常用命令

- 登录
   - mysql -u root -p
- 列出数据库
   - show databases
- 选择数据库
   - use <数据库名>
- 列出表
   - show tables
- 列出表列
   - show columns from <表名>
   - describe <表名>
- 显示服务器状态信息
   - show status
